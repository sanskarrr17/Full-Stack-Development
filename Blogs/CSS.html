CSS

CSS, or Cascading Style Sheets, is a stylesheet language to specify document presentation and design. If HTML represents the structure of a webpage, CSS represents its appearance specifically, its colors, font style, spacing, layout, and responsiveness. CSS allows developers to separate content from design which makes websites more visually appealing and much easier to maintain.
