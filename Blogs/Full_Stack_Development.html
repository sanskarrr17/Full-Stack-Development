Full Stack Development

Full stack development refers to the process of building the front-end (client - side) and back-end (server side) of the web application. A full stack developer who has knowledge and skill across the entire stack of technologies used to build a complete application

Types:

                Frontend:
                The front-end is the interface of the application, the part that users actually see and 
                design, user interface, and user experience. The core technologies are HTML 
                (Structure), CSS (Style), and JavaScript (Behaviour). Popular frameworks and 
                libraries are React, Angular, and Vue.js.  
                
                Backend:
                The back-end, this is what runs the application by handling business logic, 
                authentication, etc. When users make requests, the back-end runs business logic, 
                gets data from the database all while doing it on the back-end without users ever 
                knowing it is happening. Popular languages are Node.js, Python, Java, PHP, and 
                Ruby on Rails. API designs such as REST or GraphQL are used in communication but 
                connect the server to the client side on an application.

                Database: 
                The databases are what holds the application data organized, securely, and 
                managed. It allows an application to retrieve, update, and store data efficiently and 
                effectively. Databases can be a relational database (SQL) such as MySQL, 
                PostgreSQL, or Oracle, or they can be non-relational or NoSQL databases such as 
                MongoDB, Firebase, or Cassandra.
